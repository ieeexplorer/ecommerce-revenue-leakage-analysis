E-commerce Revenue Leakage & Retention Analysis

📊 Project Overview

This portfolio project demonstrates SQL skills, pandas data analysis, and business storytelling through actionable recommendations for an e-commerce company.

📈 Key Findings

  • Revenue: $1,115 across 15 completed orders
  • Top Category: Electronics ($515 revenue)
  • Return Rate: 20% overall
  • On-Time Delivery: 66.7%

  • 01 Data Cleaning & Analyst-Ready Dataset
  • 02 Business Analysis
  • 03 Visualizations

🗂️ Project Structure

  • sql/ - SQL queries for analysis
  • reports/ - Executive summary
  • dashboard/ - Dashboard specifications
